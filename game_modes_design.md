# 🌌 アストロフレイ・超大型拡張パック構想 (Epic Game Modes Design)

現在のアストロフレイを「単なるブラウザのドッグファイトゲーム」から、「フレンドと何時間でも遊べる超大作・マルチプレイゲーム」へと昇華させるための新モード構想です。
今後のコード実装（全体で10,000行クラスの増築）を見据え、ゲームシステム、登場アイテム、UI変更、追加されるべきプログラミング上の概念を具体的に定義しました。

---

## 🎯 モード1：アストロ・ストレンジャーレース（アイテムバトル＆レース）
宇宙での「マリオカート風」アイテム争奪・妨害型のサバイバルバトル/レースモードです。

### 📜 ルールと勝利条件
* **ルール:** 狭く障害物が多い闘技場、あるいはコース状のマップで戦います。プレイヤーの通常の射撃（メインウェポン）は威力が1/10に抑えられ、弾薬制限がかかります。
* **アイテム争奪:** マップ上のランダムな位置に「光る補給コンテナ」が一定間隔でワープアウト（出現）します。これに自機を接触させると、ランダムな「特殊武装」を1回分ストックできます。
* **勝利条件（バトル）:** 制限時間内に他プレイヤーをアイテムで撃墜したポイント（キル数）を競います。
* **勝利条件（レース）:** 指定されたチェックポイントを順番に通過し、最も早く3周したプレイヤーの勝利です。

### 📦 登場アイテム（武装）案
1. **プラズマ・ハウンド (追尾ミサイル / 赤甲羅相当)**
   * **効果:** 発射すると、最も近い敵機のエンジン熱源を感知して自動で追いかけるミサイル。直撃すると中ダメージ＋一瞬のスピン（操作不能時間）を与える。
2. **グラビティ・マイン (重力機雷 / バナナ相当)**
   * **効果:** 自機の後方に機雷を射出。機雷から半透明な重力場が発生しており、敵が近づくと強い力で中心に引き寄せられ、最終的に爆発ダメージを受ける。
3. **ハイパードライブ・マトリクス (超加速 / キノコ相当)**
   * **効果:** 使用した瞬間から3秒間、ブーストゲージを消費せずに通常の2倍の速度で移動できる。敵にぶつかると相手を弾き飛ばす。
4. **EMPカタストロフィ (全体妨害 / サンダー相当)**
   * **効果:** 出現率が極めて低いレアアイテム。発動した瞬間、自分以外の全員の画面に「SYSTEM ERROR」というノイズが走り、3秒間エンジンと射撃システムが完全にダウン（移動・攻撃不可）する。
5. **ナノドローン要塞 (バリア＆近接防御 / トリプル甲羅相当)**
   * **効果:** 3つの小型ドローンが自機の周囲を回転しながら防壁を形成する。敵の攻撃や障害物を3回まで無効化し、近づいてきた敵には自動でビームを放つ。

### 💻 プログラミング要件（必要なコード改修）
* **アイテムコンテナシステム:** プレイヤーの当たり判定とは異なる、「アイテムボックス用のエンティティ」管理機構。
* **インベントリ（UI）実装:** HUDの下部に「現在保持しているアイテム」のアイコンを表示し、使用するための専用キー（例：Fキーや右クリック）の入力フック処理を追加。
* **追尾AIの実装:** `Math.atan2` を用いて、フレーム毎にターゲットへの角度を再計算しながら飛ぶミサイル弾のクラス。

## 🛡️ モード2：マザーシップ・ウォーズ（拠点防衛＆強化・ベッドウォーズ風）
長時間の戦略的なチーム戦を楽しめる「マイクラのベッドウォーズ」風モードです。

### 📜 ルールと勝利条件
* **チーム戦:** 2〜4チーム（赤・青・緑・黄）に分れて争います。
* **マザーシップ（母艦）:** 各チームの初期スポーン位置に、HPが非常に高い巨大な「マザーシップ」が配置されています。マザーシップが破壊されない限り、プレイヤーは撃墜されても数秒後に無料でリスボーンできます。
* **鉱石（リソース）採掘:** マップ上（特に激戦区となる中央付近）に「クリスタルアステロイド」が浮遊しています。これを射撃で破壊すると、「クリスタル（資金）」がドロップします。
* **アップグレード:** クリスタルを自軍のマザーシップの領域内に持ち帰ることで、チーム全体の武器や、マザーシップの防衛システムを「購入（アップグレード）」できます。
* **勝利条件:** 敵チームのマザーシップを破壊し、リスポーンできなくなった敵プレイヤーを全滅させれば勝利。

### 💎 アップグレード要素の例
1. **マザーシップ・タレット増設 (防衛力アップ)**
   * 自軍の母艦の上下に自動迎撃レーザー砲台を設置。敵が近づく自動で迎撃してくれる。
2. **メインウェポン強化 (攻撃力アップ)**
   * チーム全員のレーザーが「初期の単発」から、「2連装」→「威力・弾速アップのプラズマライフル」へと段階的に進化する。
3. **シールドリジェネ (生存力アップ)**
   * ダメージを受けていない時間に回復するシールドの回復量と最大値が増加する。
4. **母艦リアクティブ・アーマー (母艦耐久アップ)**
   * 母艦へのダメージを30%軽減し、近接ダメージを与えてきた敵を弾き飛ばす装甲を追加する。

### 💻 プログラミング要件
* **巨大エンティティ（母艦）の衝突判定と描画:** 全長が通常の船の10倍あるエンティティのための複合的な当たり判定（複数の円形コリジョンまたは多角形コリジョン）の実装。
* **経済システムとUIバインディング:** 各プレイヤーが持つ「クレジット量」の管理と、母艦に接近したときにだけ表示される「ショップ（アップグレード）UI」の表示ロジック。
* **リスポーン動的制御:** 「母艦生存フラグ」をPhoton/Firestore等のシリアライズデータに持たせ、リスボーンの可否をサーバー同期で厳密に管理する通信処理。

## 🧟 モード3：パラサイト・パニック（感染者・サバイバル）
緊張感と恐怖を重視した、1vs多数の変則型鬼ごっこです。

### 📜 ルールと勝利条件
* **ゲーム開始:** 全員が人間の「サバイバー」としてスタートします。20秒の準備期間（逃げる時間）の後、ランダムな1人が「パラサイト（感染者）」に変異します。
* **サバイバーの目的:** 制限時間（例: 3分間）を逃げ延びるか、隠れきるのが目的。
* **パラサイトの能力と目的:** パラサイトは射撃やシールドが使えず、ブーストゲージもありませんが、**基本移動速度が人間の1.5倍**かつ**常にステルス状態（レーダーに映らない）**というチート性能を持っています。サバイバーに「体当たり」することで、ダメージ計算なしで一撃で感染（撃破）させます。
* **感染の連鎖:** パラサイトに倒されたサバイバーも瞬時にパラサイトとしてリスポーンし、人間を襲い始めます。
* **勝利条件:** 時間切れまで1人でもサバイバーが生き残ればサバイバーの勝利。全員が感染すればパラサイトの勝利。

### 💻 プログラミング要件
* **機体性能の極端なオーバーライド機能:** クラス機構を改修し、同じプレイヤーオブジェクトであっても、陣営（パラサイト）によってパラメータ、描画アルゴリズム（禍々しいオーラや血のような赤黒いエフェクト）、操作ロジックを変更する「変異（Mutate）メソッド」。
* **専用レーダーの無効化処理:** ミニマップを描画する関数おいて、対象がパラサイトの場合は距離が極端に近い（可視範囲）時しか描画しないための条件分岐。

## 🛸 モード4：ギャラクティック・ペイロード（巨大貨物護衛）
MOBAやチームシューター黎明期から人気のある「ペイロード（車や船を押す）」モードです。

### 📜 ルールと勝利条件
* **攻撃（襲撃）チームと防衛（護衛）チーム**に完全に分かれます。
* **ペイロード（輸送船）:** 防衛側の基地から、攻撃側の基地へ向かって事前に決められた「ルート（軌道）」の上をゆっくりと進む超巨大な無人輸送船がスポーンします。
* **進行条件:** 輸送船の周囲一定半径（サークル）の中に「防衛側プレイヤーのみ」が入っている間、輸送船は前進します。攻撃側プレイヤーが範囲内に入ると「Contested（争奪中）」となり停止します。
* **破壊と回復:** 攻撃側は輸送船を破壊しようと弾幕を浴びせます。護衛側が輸送船の近くにいると輸送船のHPは徐々に自動回復します。
* **勝利条件:** 時間制限内に輸送船が「最終目的地」に到着すれば防衛側の勝利。輸送船のHPを0にして完全に破壊するか、制限時間まで足止めできれば攻撃側の勝利。

### 💻 プログラミング要件
* **パス・トラッキング（自動移動）AI:** 輸送船がベジェ曲線やウェイポイント（経由点）の配列データに従って、なめらかに自動進行するためのAI計算マネージャー。
* **制圧エリア計算ロジック:** 輸送船を中心とした半径内にいるプレイヤーを毎フレーム数え上げ、アタッカーとディフェンダーの人数比から状態（前進・停止・後退・シールド回復）を計算する「テリトリー判定」関数。
* **UI表示:** 画面上部に、目的地までの進行度合いを示す専用のプログレスバーUIシステムの構築。
